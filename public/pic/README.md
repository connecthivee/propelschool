# Pic folder – PNG images & backgrounds

PNG assets used across the site:

- **kids.png** – Hero / gallery
- **kidcare.png** – Animated background (Impact section, hero subtle layer)
- **letters.png** – “Come Learn With Us” animated background
- **animation.png** – Gallery

Background images use CSS animations: `bg-pic-drift` (slow pan/scale) and `bg-pic-float` (opacity pulse). Replace these files with your own PNGs; keep filenames or update `src/lib/images.ts` and `BACKGROUNDS`.
